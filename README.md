## Overview

Thogakade Backend is a robust backend application designed to support the Thogakade system. It provides REST APIs and business logic for managing various operations with a focus on performance and scalability.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Java 8 or higher** - [Download Java](https://www.oracle.com/java/technologies/downloads/)
- **Maven 3.6.0 or higher** - [Download Maven](https://maven.apache.org/download.cgi)
- **Git** - [Download Git](https://git-scm.com/downloads)
